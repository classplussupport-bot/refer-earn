Setup Guide
Follow these steps in order for each new client
1
Create a new Google Sheet for the client
Go to sheets.google.com and create a new sheet. Copy the Sheet ID from the URL (the long string between /d/ and /edit).
2
Create 4 tabs with exact headers
Create tabs named Referrers, Leads, Payouts, Policy with these exact column headers:
Referrers: Name | Mobile | UID | PIN | TotalReferred | TotalDownloaded | TotalConverted | TotalEarned | TotalClaimed | JoinedDate

Leads: LeadID | ReferrerUID | ReferrerName | LeadName | LeadMobile | Status | ReferredDate | DownloadDate | PurchaseDate | CourseName | CoursePrice | ReferrerReward | RefereePct

Payouts: PayoutID | ReferrerUID | Name | UPI | Amount | Status | ClaimDate | PaidDate

Policy: Version | Model | ReferrerPct | RefereePct | WindowDays | MinClaim | PolicyName | Status | CreatedDate
3
Add a default Policy row
In the Policy tab row 2, add: 1 | pct | 10 | 0 | 15 | 500 | Default Policy | active | (today)
4
Register the client on the Register tab
Fill Brand Name, Org Code, Sheet ID, App ID, Password then click Register Client.
5
Share the URLs with the client
Share the Earn URL with referrers and Admin URL only with the tutor/admin.
