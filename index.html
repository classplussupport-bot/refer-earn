Setup Guide — Onboarding a New Client
Follow these steps every time you onboard a new coaching client.
1
Create a new Google Sheet for the client
Go to sheets.google.com and create a blank sheet. Name it clearly (e.g. "Bhushan Science - Refer & Earn").
Copy the Sheet ID from the URL bar — it's the long string between /d/ and /edit.
2
Add required sheet tabs with correct headers
The sheet needs 4 tabs. You can create them manually or run setupClientSheet() in the master Apps Script (change the Sheet ID first).

Referrers tab headers (Row 1):
Name | Mobile | UID | PIN | TotalReferred | TotalDownloaded | TotalConverted | TotalEarned | TotalClaimed | JoinedDate

Leads tab headers (Row 1):
LeadID | ReferrerUID | ReferrerName | LeadName | LeadMobile | Status | ReferredDate | DownloadDate | PurchaseDate | CourseName | CoursePrice | ReferrerReward | RefereePct

Payouts tab headers (Row 1):
PayoutID | ReferrerUID | Name | UPI | Amount | Status | ClaimDate | PaidDate

Policy tab headers (Row 1):
Version | Model | ReferrerPct | RefereePct | WindowDays | MinClaim | PolicyName | Status | CreatedDate
3
Add a default policy row in the Policy tab
In the Policy tab, add Row 2 with these values:
1 | pct | 10 | 0 | 15 | 500 | Default Policy | active | today's date

This sets 10% reward, 15-day window, ₹500 minimum claim.
4
Register the client using the Register tab
Go to the Register Client tab above and fill in the Brand name, Org code, Sheet ID, App ID and password. Click Register — it saves everything to the master sheet automatically.
5
Share the URLs with the client admin
After registering, copy the Admin URL and Earn URL from the output.

Admin: https://classplussupport-bot.github.io/refer-earn/admin/?org=ORGCODE
Earn: https://classplussupport-bot.github.io/refer-earn/earn/?org=ORGCODE

The admin uses the Admin URL to manage referrers, leads and payouts.
Referrers use the Earn URL to register and share their unique link.
6
Set a reward policy from the Admin dashboard
Login to the Admin dashboard and go to Reward Policy. Set the referrer reward %, referral window and minimum claim amount, then click Save & Publish.
