Setup Guide — Adding a New Client
3 simple steps — everything else is automated ⚡
1
Create a blank Google Sheet
Go to sheets.google.com → New blank sheet. Copy the Sheet ID from the URL — it's the long string between /d/ and /edit.

⚠️ Make sure the sheet is created under the same Google account that owns the Master Script. If different, share the sheet with that account giving Editor access.
2
Register the client on the Register tab
Go to the Register Client tab → fill in:

• Brand Name — client's coaching name
• Org Code — short unique code (letters/numbers, e.g. BSCI001)
• Google Sheet ID — from Step 1
• Play Store App ID — from their Play Store URL (e.g. co.classplus.abc)
• Admin Password — set a password for the admin dashboard

Click Register Client — all 4 sheet tabs are created automatically with correct headers and a default reward policy. ✅
3
Share the URLs with the client
After registering, two URLs are generated:

• Admin URL — share only with the tutor/admin. They log in to see leads, manage payouts and set reward policy.
• Earn URL — share with referrers/students. They register, get their unique referral link and track earnings.
✓
That's it — nothing else needed!
The sheet tabs, headers, and default policy are all created automatically on registration. The admin can update the reward policy anytime from their dashboard.
